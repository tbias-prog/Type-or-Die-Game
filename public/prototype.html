⚙ PROTOCOL DEPLOYMENT GUIDELINES

This is a fully-contained standalone game prototype. The JavaScript intercepts GET and POST requests made via fetch() pointing to /api/leaderboard/ and /api/submit/. When integrated into a real Django project, standard network fetch calls will naturally propagate to your endpoints!

1. TYPING ENGINE
Words spawn with increasing length. Press Space or Enter to match and discharge the particle lasers. Speed level scales automatically.
2. RETRO SOUNDS
Synthesizes retro 8-bit game audio on the fly utilizing the browser's native Web Audio API. Zero dependencies or audio assets required.
3. DJANGO BACKEND
In Django, write standard Views returning JSON highscores, and point standard forms/requests here. Fallbacks safely stream to LocalStorage.
